# 카메라 이동
```
transform.position = Vector3.Lerp(transform.position, cameraPinPoint.position, pLerp);
```
카메라가 주인공을 쫓아가게 할 때 Lerp를 쓴다.  
Lerp가 단순히 수치 변화만 시킬 때 쓸줄 알았는데 이렇게 타겟을 쫓아가게 할 때도 잘 쓰인다.
